Object Oriented Programming

Java is known as an object oriented programming lanuage.

This means that it is easy to represent entinties as object by using classes and encapsulation.

An example of this might be a Student class to represent a student 

          
            public class Student {
              /* Student propwrties */
              private string name;
              private int age;

              /* Constructor */
              public Student(String name, int age) {
                this.name = name;
                this.age = age;


              } 

              /* Getter method */
              public String getName() {
                return name;
              } 

              /* Setter method */
              public void setName(String name) {
                this.name = name;
              }
            }

he use this class by doing the following:

          
            Student student1 = new Student("Jimmy", 19);
